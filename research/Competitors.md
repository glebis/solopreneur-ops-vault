## Direct Competitors

### 1. Matt Pocock — Claude Code for Real Engineers

- **Price:** $795 for 2-week cohort
- **Audience:** Software engineers (TypeScript-heavy)
- **Content:** LLM constraints, Plan/Execute/Clear patterns, AGENTS.md, custom skills, autonomous loops
- **Strengths:** Strong personal brand (200K+ Twitter, known for TypeScript content), polished curriculum, AI Hero platform
- **Weakness:** Developer-only audience, no coverage of non-code use cases, no community/ongoing access
- **Your edge:** Non-developer focus, 6-week depth vs 2-week sprint, 50+ skills library, community

### 2. AI Product Academy — OpenClaw & Claude Code Certification

- **Price:** $2,999 (includes Mac mini hardware)
- **Audience:** Product managers
- **Content:** OpenClaw agents, Claude Code, n8n workflow automation, observability
- **Instructors:** Dmitry Shapiro (ex-CTO MySpace), Dr. Marily Nika (Google)
- **Strengths:** Industry certification, hardware included, strong PM-specific curriculum
- **Weakness:** Extremely high price, PM-only focus, less hands-on building
- **Your edge:** 5x cheaper, broader audience, more hands-on agent building, less corporate

### 3. Harold Dijkstra — AI Agent Builder Bootcamp (Maven)

- **Price:** $800 for 2.5 weeks
- **Audience:** Non-technical professionals (PM, design, marketing)
- **Content:** No-code agent building, step-by-step guidance
- **Strengths:** 1400+ bootcamp students, 30+ cohorts experience, non-technical friendly
- **Weakness:** No-code only (no Claude Code), shorter duration, no ongoing community
- **Your edge:** Claude Code (more powerful than no-code tools), 6-week depth, Russian+English markets, skills as lasting assets

### 4. Alexey Grigorev — From RAG to Agents (Maven)

- **Price:** ~$500–800 for 9 weeks
- **Audience:** Engineers transitioning to AI
- **Content:** RAG, agents, production AI systems
- **Strengths:** DataTalks.Club community (100K+ students), deep technical content
- **Weakness:** Very technical, no non-developer track, Python-heavy
- **Your edge:** Accessible to non-devs, agent-first (not RAG-first), Claude Code native

## Indirect Competitors

### Free Tier (Biggest Threat by Volume)

- **Anthropic Academy** — 17 free certified courses on Claude Code, MCP, agent skills (updated April 2026). Official material, self-paced. Threat: high-quality free baseline. **Counter:** self-paced has ~3-15% completion rate vs your 80%+ in cohort.
- **DeepLearning.AI (Andrew Ng)** — Free agentic AI course. Vendor-neutral Python approach. Massive brand. **Counter:** no hands-on projects, no feedback, no community.
- **CC for Everyone** — Free Claude Code tutorial, no coding required. **Counter:** no depth, no cohort, no support.

### Platform Competitors

- **Reforge** — $1,995/yr membership. AI strategy within broader product/growth curriculum. Enterprise-focused. **Counter:** not hands-on technical, not agent-specific.
- **Udemy** — 20+ Claude Code courses at $15–80. Largest marketplace volume (13,887 students on top course). **Counter:** low completion rates, no feedback, outdated quickly, no community.
- **Coursera** — Free Claude Code courses from Vanderbilt + Anthropic. Academic credibility. **Counter:** self-paced, no practical projects, no personal feedback.

## Positioning Gaps

### What Nobody Is Teaching

1. **Hands-on agent building for non-developers using Claude Code** — Matt Pocock targets engineers, Harold Dijkstra uses no-code tools. Nobody combines Claude Code's power with non-developer accessibility. **This is your sweet spot.**

2. **Obsidian + AI agents** — PKM + agent integration. Graham Helton writes about it, Dr. Tali Režun builds MCP for it, but nobody teaches it as a structured course. Your vault approach is unique.

3. **MCP skills as reusable assets** — Nobody teaches how to build, share, and maintain a library of 50+ skills. Skills-as-lead-magnets is an untapped positioning.

4. **Solopreneur automation with agents** — Most courses target employees at companies. Nobody specifically addresses the solo educator/creator use case with agents.

5. **Bilingual (Russian + English) AI education** — Zero competition in this niche. You're the only person with 350+ alumni in Russian who's now expanding to English.

6. **Agent-native publishing** — Using agents to maintain a public vault, auto-research, auto-draft, auto-deploy. This vault itself is the demo.
