# Conference Speaking Pipeline

> CFP → Talk → Capture → Nurture → Convert. 56% capture rate benchmark.

## The Pipeline

```
Submit CFP (3-6 months before event)
    ↓
Prepare talk (live demo + framework + CTA)
    ↓
Deliver at conference
    ↓
Capture leads (QR code → free resource)
    ↓
Nurture (email sequence + LinkedIn connection)
    ↓
Convert (community → cohort → corporate)
```

## Benchmark: 56% Capture Rate

From [[research/Partnership Channels]]: Facilitator School captured 339 signups from 600 attendees at Miro events. Key: shareable template + QR code during talk.

## Talk Format That Converts

From [[research/Partnership Channels]]:

```
0-5 min:  Contrarian hook ("Stop learning prompt engineering")
5-15 min: Live demo — build something real in 10 min
15-18 min: Framework — the mental model behind what they saw
18-20 min: CTA — QR code to free resource + community invite
```
