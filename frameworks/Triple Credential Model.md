# Triple Credential Model

> Badge + Portfolio Project + Published Skill. More powerful than any single credential.

## Why Three

From [[research/Certification Strategy]]:
- **Badges alone** — easy to get, low signal. "Completed a course."
- **Portfolio alone** — strong but invisible unless you share it.
- **Published skill alone** — proves ability but no institutional backing.

**Together:** institutional validation (badge) + demonstrated ability (portfolio) + ongoing utility (skill). Each reinforces the others.

## The Three Credentials

### 1. Digital Badge (via Certifier)
- Cost: €0 (free tier)
- Issued on lab completion
- Auto-shared to LinkedIn (drives organic visibility)
- Verifiable at `solo.salient.community/verify`
- Tiered: Graduate → Builder → Mentor

### 2. Portfolio Project
- Built during lab (weeks 3-6)
- Deployed and publicly accessible
- Documented with case study
- Demonstrates real capability, not just knowledge

### 3. Published Skill
- At least one Claude Code skill published to GitHub
- Proves ability to create reusable agent behaviors
- Others can install and use it — ongoing proof of work

## Impact on Sales

From research: 78% of professionals equate online certifications with traditional degrees. Alumni who share badges on LinkedIn drive organic enrollment. The triple credential makes each graduate a walking advertisement.
