# Value Ladder

> Free → Paid → Premium → Custom. Each step earns trust for the next.

## The Ladder

```
┌──────────────────────────────────────┐
│ Custom Consulting (€200/hr)          │  ← Highest value
├──────────────────────────────────────┤
│ Enterprise per-seat (€1,200/person)  │
├──────────────────────────────────────┤
│ Team Program (€10,000-15,000)        │
├──────────────────────────────────────┤
│ Corporate Workshop (€2,500/half-day) │  ← Compliance + capability
├──────────────────────────────────────┤
│ White-Label Delivery (€1K-1.5K/day)  │  ← Via consultancies
├──────────────────────────────────────┤
│ Lab Cohort (€950)                    │  ← Core engine
├──────────────────────────────────────┤
│ Community (€49/mo or €399/yr)        │  ← Recurring flywheel
├──────────────────────────────────────┤
│ 5-Day Challenge (€29)                │  ← Conversion event
├──────────────────────────────────────┤
│ Free Skills + Public Vault           │  ← Lead magnet, SEO
└──────────────────────────────────────┘
```

## Transition Triggers

| From | To | Trigger |
|------|-----|---------|
| Free skills | Community | "I use 3+ skills, want more + support" |
| Community | Lab cohort | "I want structured learning + cohort accountability" |
| Lab cohort | Corporate | "My team needs this too" |
| Lab cohort | Consulting | "Can you build this for us?" |

## Key Insight from Research

From [[research/Community Pricing]]: the community tier is the flywheel — each interaction is simultaneously delivery, content, social proof, and referral. It funds the free tier and feeds the cohort.
