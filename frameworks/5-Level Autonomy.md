# 5-Level Autonomy Model

> From "do this for me" to "handle everything while I sleep"

## The Levels

| Level | Name | Human Role | Agent Role | Example |
|-------|------|-----------|------------|---------|
| L1 | **Operator** | Does the work | Suggests next step | "You could post on LinkedIn now" |
| L2 | **Copilot** | Decides and acts | Drafts and prepares | Agent writes post draft, you review and publish |
| L3 | **Supervisor** | Reviews output | Executes autonomously | Agent posts, you review results daily |
| L4 | **Monitor** | Checks exceptions | Runs independently | Agent manages posting calendar, alerts on anomalies |
| L5 | **Observer** | Sets goals only | Full autonomy | "Get me 10 leads/month" — agent handles everything |

## Where to Start

Most solopreneurs should start at **L2 (Copilot)** and gradually move tasks to L3 as trust builds.

**L2 is the sweet spot because:**
- You maintain voice and judgment
- Agent does the heavy lifting (research, drafting, scheduling)
- You learn what the agent does well vs. poorly
- Easy to course-correct before damage is done

## Progression Path

```
Week 1-2:  L1-L2 for everything (learning)
Week 3-4:  L2 for content, L3 for research
Week 5-6:  L3 for content scheduling, L4 for market scans
Month 2+:  L4 for routine tasks, L2-L3 for new tasks
```

## Trust Calibration

Move up a level when:
- Agent output quality > 80% on 10+ iterations
- You stop editing agent drafts significantly
- The task is routine, not strategic

Move down a level when:
- Agent makes errors you didn't catch
- The domain changed (new market, new product)
- Stakes are high (investor pitch, major client)

## Applied to This Vault

| Task | Current Level | Target Level |
|------|--------------|-------------|
| LinkedIn comment drafts | L2 (Copilot) | L2 (keep — voice matters) |
| Lead research | L3 (Supervisor) | L4 (Monitor) |
| Market scans | L4 (Monitor) | L4 (stable) |
| Dashboard curation | L3 (Supervisor) | L4 (Monitor) |
| Content creation | L1 (Operator) | L2 (Copilot) |
