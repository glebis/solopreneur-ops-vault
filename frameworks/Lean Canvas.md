# Lean Canvas

> One-page business model. Ash Maurya's adaptation for solopreneurs.

| Block | Claude Code Lab |
|-------|----------------|
| **Problem** | Teams and individuals want to use AI agents but can't start without hand-holding. YouTube/docs aren't enough. |
| **Solution** | 6-week hands-on cohort with 50+ reusable skills, non-developer friendly, real projects from week 1. |
| **Key Metrics** | Cohort fill rate, alumni retention, community growth, MRR from membership |
| **Unique Value Proposition** | "Build with AI agents from day one — no code required. 350+ alumni, 50+ skills." |
| **Unfair Advantage** | 50+ battle-tested skills library, bilingual (RU+EN), 20 years IT + education combo, Berlin tech scene access |
| **Channels** | LinkedIn (organic), YouTube (demos), Alumni referrals, Community word-of-mouth, Public vault/site (SEO) |
| **Customer Segments** | PMs, designers, founders, analysts who need AI agents but aren't developers. Berlin/EU/global English speakers. |
| **Cost Structure** | ~€4,000/mo (rent, tools, insurance, tax). Near-zero marginal cost per student. |
| **Revenue Streams** | Cohort enrollment (€950), Community membership (€49/mo), Corporate training (custom), Consulting |

## Revenue Targets

From [[frameworks/Business Metrics]]:
- **Minimum:** €4,000/mo (covers costs)
- **Comfortable:** €7,000/mo (growth + buffer)
- **Break-even:** 5 individual students or 2 team enrollments
