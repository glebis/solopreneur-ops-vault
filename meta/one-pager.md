# Solopreneur Ops Vault — One Pager

## The Job

**When** I wake up and don't know what action to take toward selling English-language educational products,
**I want** a dashboard with max 5 clear items (one 5-min task, one agent-completed),
**so I can** take daily actions that compound into visibility and sales.

## The Pain

50 skills and products built. Zero systematic sales pipeline. Mornings lost to Telegram, YouTube, and coding yet another skill instead of revenue-generating actions. First English-language cohort undersold.

## The Switch

| Force | What's happening |
|-------|-----------------|
| **Push** | No morning clarity. Products exist but don't sell themselves |
| **Pull** | Full-screen dashboard, agents pre-did work, I collect results |
| **Habit** | Telegram/YouTube/coding fills the void |
| **Anxiety** | Task pile-up → demotivation → abandonment |

## Success Looks Like

Open vault → see 5 items → do one in 5 minutes → agent already did another → something visible ships today. Every day.

## Key Design Constraints

- Max 5 dashboard items (hard limit)
- Auto-archive stale tasks (3 days)
- One-click from insight to action
- Full-screen morning mode
