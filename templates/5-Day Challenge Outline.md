# "Build an AI Agent in 5 Days" Challenge

> €29 conversion event. Feeds the [[frameworks/Community Flywheel]].

## Overview

| | Details |
|---|---|
| **Price** | €29 (low barrier, high perceived value) |
| **Duration** | 5 days, 30 min/day |
| **Format** | Daily video + task + community discussion |
| **Goal** | Participant builds one working agent by day 5 |
| **Conversion** | Challenge → Community (€49/mo) or Lab (€950) |

## Day-by-Day Structure

### Day 1: "Your First Agent" (30 min)
- Install Claude Code
- Run first command
- Create CLAUDE.md
- **Deliverable:** Screenshot of working Claude Code session

### Day 2: "Context is Everything" (30 min)
- Complete the [[templates/Context Interview]]
- Generate context.md
- See personalized agent output
- **Deliverable:** context.md file

### Day 3: "Build a Skill" (30 min)
- Create a simple skill (e.g., daily summary generator)
- Test it 3 times
- **Deliverable:** Working skill file

### Day 4: "Agent + Tools" (30 min)
- Connect one MCP server (file system or web search)
- Agent does something it couldn't before
- **Deliverable:** Screenshot of agent using external tool

### Day 5: "Ship It" (30 min)
- Polish and document the agent
- Share in challenge community
- **Deliverable:** Published project + community post

## Conversion Moment

After Day 5:
- "Want to go deeper? Join the community for ongoing support (€49/mo)"
- "Ready for the full experience? Next lab cohort starts [date] (€950)"

Expected conversion: 10-15% to community, 3-5% to cohort.

## Revenue Math

50 participants × €29 = **€1,450 per challenge**
+ 5 community conversions × €49/mo × 6 months = **€1,470**
+ 2 cohort conversions × €950 = **€1,900**
**Total per challenge: ~€4,820**

Run 4/year = **€19,280/yr**
