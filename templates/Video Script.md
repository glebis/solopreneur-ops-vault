# Video Script Structure

## For skill demos (2-5 min)

```
0:00 — Hook: "Here's what happens when you [action]"
0:15 — Context: What problem this solves (15 sec max)
0:30 — Demo: Show the skill in action (main content)
X:XX — Result: Show the output
X:XX — CTA: "If you want to learn this, link in description"
```

## For teaching videos (5-15 min)

```
0:00 — Hook: Contrarian statement or surprising result
0:30 — Setup: Why this matters (30 sec)
1:00 — Framework: The mental model (with visual)
2:00 — Example 1: Concrete walkthrough
5:00 — Example 2: Variation or edge case
8:00 — Common mistakes: What NOT to do
10:00 — Summary: 3 key takeaways
11:00 — CTA: Subscribe + cohort mention
```

## Production Checklist

- [ ] Talking head with screen recording
- [ ] Captions added (2.3x engagement boost)
- [ ] Thumbnail prepared ([[templates/Video Thumbnail]])
- [ ] Description with timestamps
- [ ] Tags and keywords from [[research/YouTube Strategy]]
