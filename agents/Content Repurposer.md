# Content Repurposer

> 1 input → 7+ outputs. Agent handles the decomposition.

## What It Does

Takes a lab session recording/transcript and produces platform-specific content:

1. YouTube description + timestamps
2. LinkedIn post (key insight + hook)
3. Blog post outline
4. 3 social clips (60s highlights)
5. FAQ additions from Q&A
6. Testimonial extraction
7. Skill demo identification

## Autonomy Level

**L2 (Copilot)** — agent drafts all outputs, human reviews for voice and accuracy before publishing.
